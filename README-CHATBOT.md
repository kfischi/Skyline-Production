## 🚀 התקנה - שלב אחר שלב

### שלב 1: התקן את הספרייה

פתח טרמינל בתיקיית הפרויקט והרץ:

```bash
npm install @google/generative-ai
```

**✅ זה מתקין את ספריית Google Gemini**

---

### שלב 2: קבל מפתח API

1. **גש לאתר:**
   ```
   https://makersuite.google.com/app/apikey
   ```

2. **התחבר עם חשבון Google**

3. **לחץ על "Create API Key"**

4. **העתק את המפתח** - נראה כך:
   ```
   AIzaSyD...........................
   ```

**⚠️ שמור את המפתח במקום בטוח!**

---

### שלב 3: צור קובץ .env.local

בתיקייה **הראשית** של הפרויקט (ליד `package.json`), צור קובץ חדש בשם:

```
.env.local
```

**הדבק את התוכן הזה:**

```bash
GEMINI_API_KEY=המפתח_שלך_כאן
```

**החלף את `המפתח_שלך_כאן` במפתח האמיתי שקיבלת!**

**דוגמה:**
```bash
GEMINI_API_KEY=AIzaSyD1234567890abcdefghijk
```

---

### שלב 4: בדוק שהקבצים קיימים

ודא שהקבצים הבאים קיימים בפרויקט:

```
✅ /app/api/chat/route.js
✅ /components/layout/ChatBot.jsx
✅ /components/layout/ChatBot.module.css
✅ /app/layout.jsx (מעודכן)
✅ /package.json (מעודכן)
✅ /.env.local (יצרת עכשיו!)
```

---

http://localhost:3000

## ⚠️ בעיות נפוצות

### ❌ "Failed to fetch" / שגיאת רשת

**פתרון:**
1. ודא ש-`.env.local` קיים
2. ודא שהמפתח נכון (ללא רווחים)
3. אתחל את השרת (`Ctrl+C` ואז `npm run dev`)

### ❌ "API key not valid"

**פתרון:**
1. בדוק שהמפתח התחיל ב-`AIza...`
2. צור מפתח חדש באתר של Google
3. ודא שאין רווחים לפני/אחרי המפתח

### ❌ הכפתור לא מופיע

**פתרון:**
1. בדוק שהוספת `<ChatBot />` ל-`layout.jsx`
2. נקה cache: `rm -rf .next` ואז `npm run dev`

## 🆘 עזרה

### זקוק לעזרה?

1. **בדוק את ה-Console:**
   - פתח Developer Tools (`F12`)
   - לחץ על "Console"
   - חפש שגיאות אדומות

2. **בדוק את ה-Network:**
   - פתח Developer Tools
   - לחץ על "Network"
   - שלח הודעה לבוט
   - חפש request ל-`/api/chat`

3. **אתחל הכל:**
   ```bash
   rm -rf .next
   rm -rf node_modules
   npm install
   npm run dev
   ```
